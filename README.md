# Fan Tracks for Heat: Pedal to the Metal

Unofficial fan-made tracks for the board game *Heat: Pedal to the Metal*.

## Tracks
- Brazil GP
- Monaco GP

## Printing
Print at 100% scale on A4 paper.

## Disclaimer
This is an unofficial fan project for personal and community use ONLY!  
Heat: Pedal to the Metal is published by Days of Wonder / Asmodee.
